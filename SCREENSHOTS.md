# 界面截图和功能展示

本文档展示 iFlow SDK Web Interface 的主要界面和功能。

## 🖥️ 主界面

### 聊天界面
```
主界面包含三个主要区域：
1. 左侧边栏 - 显示连接状态、工具调用和任务计划
2. 中央聊天区 - 显示对话历史和消息输入框
3. 配置面板 - 点击右上角设置图标打开
```

**特点:**
- ✨ 现代化的 UI 设计
- 💬 流式消息显示（打字机效果）
- 📝 Markdown 渲染支持
- 🎨 美观的消息气泡
- ⏱️ 相对时间显示

### 欢迎屏幕
```
┌─────────────────────────────────────────┐
│                                          │
│              🤖                          │
│                                          │
│        欢迎使用 iFlow SDK                │
│                                          │
│       开始与 AI Agent 对话吧             │
│                                          │
└─────────────────────────────────────────┘
```

当没有消息时显示友好的欢迎界面。

## 🎯 核心功能展示

### 1. 实时对话

**用户消息 (右侧，蓝色气泡):**
```
┌─────────────────────────────────────────┐
│                          [👤]            │
│                    你好，介绍一下自己    │
│                       3 分钟前           │
└─────────────────────────────────────────┘
```

**AI 响应 (左侧，白色气泡):**
```
┌─────────────────────────────────────────┐
│  [🤖]                                    │
│  你好！我是 AI 助手，基于 iFlow SDK...   │
│  我可以帮你：                             │
│  • 回答问题                              │
│  • 执行工具操作                          │
│  • 规划复杂任务                          │
│  2 分钟前                                │
└─────────────────────────────────────────┘
```

### 2. 代码高亮显示

```python
# AI 返回的代码会自动高亮
def fibonacci(n):
    if n <= 1:
        return n
    return fibonacci(n-1) + fibonacci(n-2)
```

### 3. 工具调用监控

**左侧边栏 - 工具调用区域:**
```
🔧 工具调用
├─ 文件操作           [✅ 完成]
│  创建文件 test.txt
│
├─ 网络请求           [🔵 运行中]
│  获取天气数据
│
└─ 代码执行           [⚪ 等待]
   执行 Python 脚本
```

### 4. 任务计划可视化

**左侧边栏 - 任务计划区域:**
```
📋 任务计划

任务: 创建 Web 应用
├─ 🟢 设计数据库结构
├─ 🔵 实现后端 API
└─ ⚪ 创建前端界面
```

### 5. 配置面板

**权限模式选择:**
```
⚙️ 配置

🛡️ 权限模式
  ◉ 自动 (Auto)
     自动批准所有工具调用
  
  ○ 确认 (Confirm)
     每次工具调用前需要确认
  
  ○ 拒绝 (Deny)
     禁止所有工具调用
```

**进程管理:**
```
▶️ 进程管理
  ☑ 自动启动 iFlow 进程
     SDK 会自动启动和管理 iFlow CLI 进程
```

**会话管理:**
```
#️⃣ 会话 ID (可选)
  [输入框: 留空以创建新会话]
  
  输入现有会话 ID 以继续之前的对话
```

## 🎭 主题和样式

### 颜色主题

**主色调:**
- 主蓝色: `#0ea5e9` (primary-500)
- 深蓝色: `#0369a1` (primary-700)

**状态颜色:**
- 成功: `#10b981` (绿色)
- 警告: `#f59e0b` (黄色)
- 错误: `#ef4444` (红色)
- 信息: `#3b82f6` (蓝色)

**中性色:**
- 背景: `#f9fafb` (gray-50)
- 边框: `#e5e7eb` (gray-200)
- 文本: `#111827` (gray-900)

### 字体

- 主字体: Inter, system-ui
- 代码字体: Monaco, Consolas, monospace
- 基础字号: 16px
- 行高: 1.5

### 阴影

- 轻微: `shadow-sm`
- 中等: `shadow-md`
- 重度: `shadow-lg`
- 超重: `shadow-xl`
